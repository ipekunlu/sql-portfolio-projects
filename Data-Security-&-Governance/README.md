# Data Security & Governance (PostgreSQL)

## Overview
Practical security exercises demonstrating role-based access control, row-level security (RLS), and privilege auditing in PostgreSQL.

## What's included
- RLS implementation and policy testing using roles
- Privilege audit report for governance documentation
- Security Q&A notes (concept recap)

## Files
- Row_Level_Security_Implementation.sql -> Roles, permissions, RLS policies (passwords omitted for portfolio safety)
- User_Privileges_Audit_Report.docx -> Privilege audit summary
- Data_Security_Q&A.docx -> Security concepts and Q&A notes

## Skills Demonstrated
- GRANT/REVOKE and role design
- Row-Level Security (RLS)
- Security auditing and documentation
