# SQL Analytics Projects
# Based on the PostgreSQL sample 'sh' (Sales History) schema

## Overview
This folder contains analytical SQL projects focused on sales performance, market trends, and KPI reporting.  
The work demonstrates practical business-oriented data analysis using PostgreSQL and advanced SQL techniques.

## Projects

### Sales Performance - Window Functions
This project analyzes sales data using window functions to identify trends and performance patterns.

Key tasks include:
- Calculating running totals and moving averages
- Comparing performance across time periods
- Using LAG and window frame modes (ROWS, RANGE, GROUPS)

File:
- Sales_Performance_Window_Functions.sql


### Market Trend and KPI Reporting
This project focuses on business analytics and reporting tasks used in real-world decision-making.

Key tasks include:
- Customer sales contribution analysis
- Monthly and yearly KPI calculations
- Identifying top-performing customers and regions
- Sales trend analysis over time

File:
- Market_Trend_and_KPI_Reporting.sql


## Skills Demonstrated
- SQL analytics
- Window functions
- KPI reporting
- Ranking and aggregation
- Data analysis for business insights

## Note: Requires the tablefunc extension to be enabled in PostgreSQL (CREATE EXTENSION tablefunc;).
